# AND-22 Panex adversarial product dogfood

## Rationale

We intentionally made AND-22 BDD-heavy and TDD-light because the remaining risk is Панэкс product behavior under realistic user-style requests, not backend API transport. The slice adds five adversarial scenarios and deterministic guardrails while leaving final usefulness as a human review judgment.

Full evaluator run passed 11 scenarios with 0 failures. Local tests passed: 34 focused tests and 79 targeted Agent Context tests with 2 existing warnings.
