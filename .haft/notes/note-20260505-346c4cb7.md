# Keep live smoke local by default; require explicit --api-url for Fly

## Rationale

AND-15 needs production/Fly smoke, but the user explicitly wants Codex/Claude to avoid accidental Panel calls. The helper therefore ignores ambient AGENT_CONTEXT_API_URL in default mode, starts local backend only by default, and switches to external target_mode only when --api-url is supplied. This preserves safe local dogfood while still enabling intentional production verification.

## Evidence

Targeted BDD tests added for explicit external mode, local default ignoring AGENT_CONTEXT_API_URL, missing external token failure, and sanitized target_mode reports.
