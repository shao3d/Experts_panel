# Raise Agent Context budgets for all-expert source bundles

## Rationale

The paid refat+akimov smoke proved 420s/5MB is enough for a two-expert case, but AgentContextService currently processes selected experts sequentially and all-expert source_bundle requests are naturally long and bulky. The explicit-only design makes a larger bounded research budget acceptable, while still failing closed if a request exceeds it. Defaults were raised to 3600s and 100000000 bytes, with CLI/smoke defaults and docs aligned.

## Evidence

Targeted Agent Context suite passed: 54 passed, 2 warnings. git diff --check clean. Paid smoke evidence from prior run remains source_bundle_valid for refat=63, akimov=57, response_bytes=1225738, processing_time_ms=261407.
