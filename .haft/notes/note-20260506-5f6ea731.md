# AND-21 Panex compact default tuning

## Rationale

Production dogfood showed Панэкс answers were useful but too long by default, weak-signal answers did not always offer targeted expansion, and the evaluator under-scored lean source_expand passports. We chose instruction-level tuning plus mode-aware evaluator checks rather than changing backend retrieval/digest behavior, because the issue is final subagent packaging, not API transport or evidence quality.

## Evidence

Local tests passed: 31 passed for subagent contract/dogfood/product-quality eval; old six-scenario dogfood re-eval passed with expand_sources_followup score improving to 0.970; broader Agent Context contour passed 76 tests with 2 existing warnings.
