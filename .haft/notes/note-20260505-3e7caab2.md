# Agent Context external links are references-only in default source_bundle

## Rationale

V1 should surface URLs that experts intentionally included in selected posts, but it must not change the trust boundary or latency profile by automatically fetching, crawling, cloning, or summarizing external resources. The response now exposes per-source external_links with fetch_status=not_fetched so subagents can cite author-supplied references and a future explicit enrichment mode can add verified summaries later.

## Evidence

BDD/TDD red-green: external_links assertions failed before implementation with KeyError, then passed. Full Agent Context suite passed after live edge-case hardening: 61 passed, 2 warnings. Local live dogfood smoke for query `vLLM LiteLLM Codex CLI GPU servers LLM inference API` and expert `neuraldeep` passed through the CLI with `source_bundle_valid`, `selection_used.use_super_passport=true`, 48 selected sources, 935912 response bytes, and 40235ms processing time. Follow-up live API inspection found 40 `external_links` across 11 selected sources, all with `fetch_status=not_fetched`, link types `github_repo`, `web`, and `telegram_post`, and `bad_suffix_links_count=0`.

Production closeout: commit `f1207b4` was pushed to `main`, GitHub Deploy to Fly.io run `25398833141` completed successfully, and Fly app `experts-panel` reached version `338` with image `experts-panel:deployment-01KQWV8DHCFVWBFHVSMD00MGEP`. Local CLI smoke to the production public URL failed with `HTTP 403` because the local env token is not the production token. Production public endpoint verification was then executed from the Fly machine against `https://experts-panel.fly.dev/api/v1/agent/context` using `AGENT_CONTEXT_API_TOKEN` from Fly env without printing the secret. The production response returned `mode=source_bundle`, `selection_used.use_super_passport=true`, `selected_source_counts.neuraldeep=24`, `sources_with_external_links=23`, `external_links_count=99`, `bad_suffix_links_count=0`, `fetch_status_counts.not_fetched=99`, `link_type_counts={github_repo:22, web:37, telegram_post:37, video:3}`, `pipeline_used_contains_external_link_references=true`, and `processing_time_ms=28321`.
