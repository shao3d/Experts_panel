#### Installation

**Backend:**


**1. Start Backend:**
```bash
cd backend

```

**2. Start Frontend:**
```bash

npm run dev
```

**3. Access Application:**
- Frontend: http://localhost:3001
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
